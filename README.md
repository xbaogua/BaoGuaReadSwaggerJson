# BaoGuaReadSwaggerJson
告别手动翻阅：一键解析 Swagger.json文件

双击运行程序，输入swagger.json文件的URL，在本机浏览器打开，本机浏览器记得挂BurpSuite的代理，这样请求测试包就能在BurpSuite中查看和修改，无需拷贝来拷贝去，非常方便。
